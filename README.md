Paso #1 Crea el entorno virtual 

	python -m venv venv

paso #2 instalar Django 

	pip install "Django==4.2.*"

paso #3 implementar Django en el proyecto 

	django-admin startproject juegomichi .

paso #4 como ejecutar el proyecto (tiene que ser en el entorno virtual)

	python manage.py runserver 

	si lo queremos en un puerto especifico 

	python manage.py runserver 3000

pip install pymysql
pip install django-cors-headers
